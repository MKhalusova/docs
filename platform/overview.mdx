Welcome to the Unstructured Platform User Guide. This guide provides comprehensive instructions and insights for users navigating the Unstructured Platform.

Its key functionalities are enhanced with [chunking strategies](/open-source/core-functionality/chunking), [embedding generation](/open-source/core-functionality/embedding) for RAG, and compatibility with multiple file types (text, images, PDFs, and many more).
